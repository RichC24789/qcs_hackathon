# Care micro-content topics — summary and format map

This file lists all 50 topics, their theme, primary content format, and secondary content format.

Each topic has a corresponding markdown file in the `/topics` folder named `topic-XX-[slug].md`.

---

## Format key

| Format | Description |
|---|---|
| Animation | 60–90 second step-by-step visual sequence. Use Lottie animations from LottieFiles.com or CSS/JS for hackathon builds. |
| Micro podcast | 3–5 minute audio episode. Two-person conversational format. Use ElevenLabs free tier to generate from script. |
| Poster | Printable A4 visual reference card. Build as styled HTML with print stylesheet. |
| Text card | Core written content card. Readable in under 2 minutes. Baseline format for every topic. |
| Scenario quiz | 2–3 scenario-based questions with immediate feedback. No backend needed — pure HTML/JS. |
| Quick reference card | Downloadable checklist or reference sheet. Build as styled HTML. |

## Topic file structure

Each topic file contains:

1. **Title and metadata** — topic number, theme, primary and secondary format
2. **Hook** — one sentence connecting the topic to frontline experience
3. **Key points** — 3–5 standalone actionable bullets
4. **Quick decision prompt** — if/then scenarios for common situations
5. **Common mistakes** — 2–3 things staff frequently get wrong
6. **Who to tell** — escalation chain with specific contacts
7. **Format-specific notes** — production guidance for the primary and/or secondary format
